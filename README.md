# Requirements

- [Vagrant](https://www.vagrantup.com/)

# Run instructions (development)

First time running:

```bash
$ vagrant up
$ vagrant ssh
vagrant@vagrant$ sudo /vagrant-src/kickstart.sh
```

Once VM is running:
```bash
$ vagrant destroy -f; vagrant up; vagrant ssh
vagrant@vagrant$ sudo /vagrant-src/kickstart.sh
```
